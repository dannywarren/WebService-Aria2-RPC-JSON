=head1 NAME

WebService::Aria2::RPC::JSON - Control an aria2 instance via the RPC interface using JSON::RPC

=head1 SYNOPSIS

  use WebService::Aria2::RPC::JSON;

  my $aria2 = WebService::Aria2::RPC::JSON->new();

  my $gid = $aria2->add_uri( 'http://example.com/file.tgz' );
  $aria2->pause( $gid );

=head1 DESCRIPTION

Control an aria2 instance via the RPC interface using JSON::RPC.

See: http://aria2.sourceforge.net/manual/en/html/aria2c.html#rpc-interface
